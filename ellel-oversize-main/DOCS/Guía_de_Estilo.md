# Guía de Estilo: Oversize. Premium Store

## Paleta de Colores

- **Fondo Principal**: `#0a0512` (Negro violáceo profundo)
- **Fondo Secundario**: `#140b24` (Para tarjetas y modales)
- **Texto Principal**: `#ffffff` (Blanco puro para contraste alto)
- **Texto Secundario**: `#b3a8c4` (Gris violáceo para párrafos)
- **Acento Primario**: `#8a2be2` (Deep Purple - Botones principales, hover effects)
- **Acento Secundario**: `#ffb703` (Amarillo Vibrante - Detalles, badges de "Nuevo", estrellas de reseñas)
- **Acento Terciario**: `#fb8500` (Naranja Cálido - Alertas, llamadas a la acción secundarias)

## Tipografía

- **Fuentes Principales**:
  - Encabezados: `Playfair Display` (Serif, elegante para hero text y logotipo).
  - Cuerpo / Interfaz: `Outfit` (Sans-serif moderno, geométrico, excelente legibilidad).

## Diseño de Referencia
- **Estilo**: "Dark Mode Premium"
- **Componentes UI**: Glassmorphism suave (fondos translucidos con `backdrop-filter: blur`), bordes muy tenues.
- **Layout**: Uso intenso de espacio en blanco (espaciado generoso) para resaltar las imágenes de los productos.
- **Micro-interacciones**: Transiciones fluidas en botones e imágenes. Hover en cards hace que el producto resalte utilizando `transform: scale()`.
